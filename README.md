# Проект банковского виджета
## Контекст проекта 
Выполнения домашних заданий второго курса 
*IT-отдел крупного банка делает новую фичу для личного кабинета клиента.*
*Это виджет, который показывает несколько последних успешных банковских операций клиента.*
*Вам доверили реализовать этот проект, который на бэкенде будет готовить данные для отображения в новом виджете.*

Проект содержит функции:
'''mask_account_card'''
функция обработки информации о картах и счетах
'''get_date'''
функция форматирования типа строки даты
''' get_mask_card_number'''
Функция принимает на вход номер карты в виде числа и возвращает маску номера по заданному правилу
'''get_mask_account'''
Функция принимает на вход номер счета в виде числа и возвращает маску номера по заданному правилу
'''filter_by_state'''
Функция возврата информации по выполненным или не выполненым операциям
'''sort_by_date'''
Функция сортировки операций по дате
