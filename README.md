# Проект банковского виджета
## Контекст проекта 
Выполнения домашних заданий второго курса 
*IT-отдел крупного банка делает новую фичу для личного кабинета клиента.*
*Это виджет, который показывает несколько последних успешных банковских операций клиента.*
*Вам доверили реализовать этот проект, который на бэкенде будет готовить данные для отображения в новом виджете.*

Проект содержит функции:
1. Функция обработки информации о картах и счетах
```mask_account_card```
2. функция форматирования типа строки даты
```get_date```
3. Функция принимает на вход номер карты в виде числа и возвращает маску номера по заданному правилу
```get_mask_card_number```
4. Функция принимает на вход номер счета в виде числа и возвращает маску номера по заданному правилу
```get_mask_account```
5. Функция возврата информации по выполненным или не выполненым операциям
```filter_by_state```
6. Функция сортировки операций по дате
```sort_by_date```
## Тестирования 
Тесты выполненны для функций с применением фикстур и параметризации.
Согласно Coverage report,проект покрыт тестами на 100%, для тестирования используется pytest. 
Для запуска тестов выполните команду pytest в корневой папке проекта.
